Detailed Description

The AptDatCache class provides access to cached information on airports from an apt.dat file.

Instead of actually caching airport information in memory, this class stores the position in the apt.dat file where each airport's record begins.

std::streamsize PPL::detail::AptDatCache::findAirport ( const std::string &  code)

Returns the position of the entry in the apt.dat file for the airport with the requested code.

Timing information: Before airports are cached, this method searches linearly throught the apt.dat file for the requested airport. This can take seconds or tens of seconds.

To find every airport in advance, call startFindingAllAirports().

Parameters
code
Returns
The position in the file before the beginning of the airport's record
Exceptions
NoSuchAirportExceptionif no airport with the requested ID exists
