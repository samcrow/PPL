Detailed Description

The GraphicsContext class provides an interface that code can use to draw simple graphics in a way that does not depend on OpenGL or other low-level interfaces.

A GraphicsContext defines an area in which graphics should be drawn. It is bounded by the top(), bottom(), left(), and right() dimensions.
