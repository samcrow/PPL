Detailed Description

template<typename T>
class PPL::OwnedData< T >

Data owned by the plugin and published as a XPLM DataRef.

create an instance and call a register function to make data accessible to other plugins

create owned date for sharing.

the identifier provided is the string identifier later used for the dataref. the dataref is then registered in X-Plane's plugin system Please consult the XP SDK naming conventions on how to name things right.

Parameters
std::string&identifier
RWTypeset the dataref writeable to OTHER plugins
publish_in_dreby default, a message is sent to the DataRefEditor plugin that exposes the dataref
callbackoptional callback function pointer to be called when other plugins write to the dataref
Exceptions
throwsDataRefNotPublishedException if publishing in X-Plane fails
