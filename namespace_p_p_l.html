enum PPL::RWType

RWType distinguishes the three types of acess to datarefs.

For simdata, it specifies if incoming data may be written to X-Plane and if data should be readable. For owned data, it specifies if the data is readable or writebale to OTHER plugins (or anything that accesses datarefs e.g. panels).
