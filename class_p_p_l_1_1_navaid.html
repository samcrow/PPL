Detailed Description

Represents a navaid of one of the following types:

  • Airport
  • NDB
  • VOR
  • ILS
  • Localizer
  • Glideslope
  • Outer marker
  • Middle marker
  • Inner marker
  • Fix
  • DME
  • Latitude/longitude (used only in the FMS)
