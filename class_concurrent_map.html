Detailed Description

template<typename K, typename V, class Map = std::unordered_map<K, V>>
class ConcurrentMap< K, V, Map >

Maps keys to values using an underlying map and provides synchronization on operations.

This class is thread-safe. Each method appears to execute atomically.

Parameters
Kthe key type
Vthe value type
MapThe type of underlying map to use
